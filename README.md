# Jadoo - Travel Agency Landing Page

A modern, responsive flight booking webpage designed with Figma and built with Tailwind CSS and HTML. This project showcases a beautiful travel agency interface for discovering and booking trips around the world.

## 🌍 Project Overview

Jadoo is a professional travel agency landing page that provides users with an intuitive interface to explore destinations, compare flight deals, and book their next adventure. The design emphasizes user experience with a clean, modern aesthetic and smooth navigation flow.

**Live Demo:** [https://travel-agency-landing-page-lemon.vercel.app](https://travel-agency-landing-page-lemon.vercel.app)

## ✨ Features

- **Header Navigation** - Clean navigation bar with language selection and sign-up options
- **Hero Section** - Eye-catching hero area with call-to-action buttons and promotional imagery
- **Service Categories** - Showcase of key services:
  - Calculated Weather
  - Best Flights
  - Local Events
  - Customization
- **Top Destinations** - Curated list of popular travel destinations with pricing and trip duration
- **Easy Booking** - Simple 3-step booking process for planning trips
- **Trip Planning** - Interactive trip planning section with progress tracking
- **Testimonials** - Customer reviews and feedback section
- **Partner Logos** - Display of trusted airline and travel partners
- **Newsletter Subscription** - Email subscription for travel updates and offers
- **Footer** - Comprehensive footer with company links and social media integration

## 🛠️ Tech Stack

- **HTML** - Semantic markup and page structure
- **Tailwind CSS** - Utility-first CSS framework for styling (57.5% of codebase)
- **Responsive Design** - Mobile-first approach with media queries

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher) - for Tailwind CSS CLI
- Basic knowledge of HTML and CSS

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/DusuTimothy/Travel-Agency-landing-page.git
   cd Travel-Agency-landing-page
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Build Tailwind CSS** (if modifying styles)
   ```bash
   npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
   ```

4. **Open in browser**
   - Simply open `index.html` in your web browser, or
   - Use a local development server like Live Server for hot reload

## 📱 Responsive Design

The landing page is fully responsive and works seamlessly across:
- Desktop (1024px and above)
- Tablet (768px - 1023px)
- Mobile (below 768px)

## 🎨 Design Inspiration

This project was designed using Figma, translating professional design mockups into a fully functional, responsive webpage. The design prioritizes:
- User engagement with vibrant colors and clear call-to-action buttons
- Information hierarchy for easy scanning and navigation
- Visual consistency throughout the page
- Accessibility with semantic HTML structure
